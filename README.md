# ATデザイン｜台本操作型 Web Slides

営業商談用のWebスライドです。発表者ビュー（台本あり）と顧客表示（スライドのみ）を分離しています。

## 使い方

1. `index.html` を開く
2. 右上の「顧客表示を開く」を押す
3. Web会議では顧客ウィンドウだけを画面共有する
4. 手元の発表者ビューで台本を見ながら進行する

## 操作

- `→` / `Space`：次へ
- `←`：前へ
- `1`〜`9` / `0`：スライド移動
- `F`：全画面

## 編集

- スライド画像：`images/01.png`〜`images/10.png`
- 台本：`slides.json`

## Vercel

Framework Preset は `Other`、Build Command なし、Output Directory なしでデプロイできます。
