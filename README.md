# ATデザイン｜販売プレゼン（台本操作型 Web Slides）

専門サービス向けの一次商談用Webスライド。発表者ビュー（台本あり）と顧客ビュー（スライドのみ）を分離しています。

**スライドは画像ではなくHTMLで描画**しています。文言の修正は `slides.js` を直すだけで反映され、投影してもベクターで鮮明です。

## 使い方

1. `index.html` を開く
2. 右上の「顧客表示を開く」を押す
3. Web会議では顧客ウィンドウだけを画面共有する
4. 手元の発表者ビューで台本を見ながら進行する

## 操作

- `→` / `Space`：次へ
- `←`：前へ
- `1`〜`9` / `0`：スライド移動
- `F`：全画面

## 編集

- 各スライドの本文・台本：`slides.js`（`window.SLIDES` 配列）
- 配色・レイアウト：`styles.css`（`:root` のトークンで色を一括変更可）
- スライドは1280×720のキャンバスで設計し、画面サイズに自動フィット
